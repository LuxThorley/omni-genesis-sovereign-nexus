## Deployment — GitHub → Cloudflare Pages (Custom Domain)

1. Push this repository to your GitHub account (already created at https://github.com/LuxThorley/omni-genesis-sovereign-nexus).
2. Cloudflare Dashboard → Workers & Pages → Create → Pages → Connect to Git.
3. Select the repository.
4. Build settings:
   - Framework preset: **None**
   - Build command: *(empty)*
   - Build output directory: `/`
5. Deploy. Cloudflare automatically enables the optional `/api/token` Pages Function.
6. Add your custom domain under Custom domains. Free SSL is issued automatically.
